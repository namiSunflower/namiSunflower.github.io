Website Name and Domain Name

The site name is entitled, "Weather Checker", while the domain name is called: "weatherchecker.news."

Site Purpose

The purpose of this website is to present information for the current weather report in the state of Idaho, specifically around the Southeastern region. It aims to provide updates of regular weather forecasts and advisories from accurate meteoroligical data. This is to help ensure every American living in the area is safe from potential hazards.
